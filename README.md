# Informo 

A ruby gem for retrieving info from linux systems.

## Installation

```
$ gem install informo
```

## Usage

```
require 'informo'
```

The following top level classes exist:

- Informo::Bios
- Informo::Dpkg
- Informo::Memory
- Informo::Network
- Informo::Processor
- Informo::Rpm
- Informo::Storage
- Informo::System
