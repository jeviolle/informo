# Informo 

A ruby gem for retrieving info from linux systems.

## Installation

clone this repo

```
$ gem build informo.gemspec
$ gem install informo-1.0.gem
```

## Usage

```
require 'informo'
```

The following top level classes exist:

- Informo::Bios
- Informo::Dpkg
- Informo::Memory
- Informo::Network
- Informo::Processor
- Informo::Rpm
- Informo::Storage
- Informo::System
